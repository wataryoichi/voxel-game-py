# Minecraft-style Voxel Game

A simple Minecraft-inspired voxel game built with Python, Pygame, and OpenGL.

## Requirements

- Python 3.10+
- macOS (tested on desktop)

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Controls

- WASD: Move
- Mouse: Look around
- Left click: Break block
- Right click: Place block
- 1-4: Select block type (grass, dirt, stone, wood)
- Space: Jump
- ESC: Quit
